# Internship SoftPrim Technology SRL

Acest repo conține exercițiile tehnice pentru pozițiile de stagiu de la **SoftPrim Technology SRL**.

Fiecare poziție are propriul subdirector, cu enunțul complet și fișierele de pornire necesare.

---

## Structură

```
internship-softprim/
├── backend/      -- exercitiu pentru pozitia Backend Developer
├── fullstack/    -- exercitiu pentru pozitia Full-Stack Web Developer
└── README.md     -- fisierul curent
```

---

## Pentru candidați

1. Identifică în anunțul de stagiu poziția pentru care ai aplicat.
2. Deschide subdirectorul corespunzător și citește integral fișierul `README.md` din acel director — acolo găsești enunțul complet al exercițiului.
3. Rezolvă cerința și trimite rezultatul conform instrucțiunilor din enunț.

Termenul de livrare și modalitățile prin care poți trimite rezolvarea sunt explicate în README-ul fiecărui exercițiu.

---

## Pozițiile deschise

| Poziție | Director | Stack principal |
|---------|----------|-----------------|
| Backend Developer | [`backend/`](./backend/) | PHP / Node.js + MySQL |
| Full-Stack Web Developer | [`fullstack/`](./fullstack/) | PHP / Node.js + MySQL + HTML / React / Vue |

Detalii despre fiecare poziție, condițiile de eligibilitate și procesul de selecție le găsești în anunțul de stagiu publicat pe platforma UPB.

---

## Despre companie

**SoftPrim Technology SRL** este o companie românească specializată în dezvoltare web, aplicații e-commerce și servicii IT pentru clienți din zona industrială și comercială.
